# Racespot.tv Website Mockup - Vercel Deployment

## Quick Start (2 Minuten)

### Option 1: Git + Vercel (Empfohlen)
```bash
# 1. Diesen Ordner in ein Git-Repo pushen
git init
git add .
git commit -m "Racespot website mockup"
git push -u origin main

# 2. Auf Vercel.com anmelden/einloggen
# 3. "Add New Project" → GitHub-Repo wählen
# 4. Deploy! ✨
```

### Option 2: Drag & Drop auf Vercel
```bash
# 1. Zip-Datei erstellen:
zip -r racespot-mockup.zip .

# 2. Auf https://vercel.com/new/import-project
# 3. "Continue with Git" → Den ZIP-Ordner hochladen
# 4. Deploy! ✨
```

## Was ist drin?

- **index.html** — Kompletter Click Dummy mit allen 7 Seiten
- **DESIGN_SYSTEM.md** — Brand System (Farben, Typography, etc.)
- **COMPONENT_LIBRARY.md** — Alle UI-Komponenten dokumentiert
- **WIREFRAMES.md** — Detaillierte Wireframes (ASCII)
- **LAYOUT_SPECS.md** — Grid, Responsive, Performance, Accessibility
- **PROJECT_BRIEF.md** — Original-Projektbeschreibung
- **vercel.json** — Vercel-Konfiguration (static site)
